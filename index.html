Learn Web Html Css And JavaScript

ditulis oleh Xie pada 7 Januari 2024

Website ini dibuat untuk belajar dan mengolah yang sudah di pelajari seperti HTML dasar, CSS dasar, CSS Layouting, CSS3 dan sebagian kecil CSS Unit. Kekurangan pada web ini tidak menggunakan JavaScript atau lainnya. Karena penulis belum mempelajari hal tersebut. Maka penulis membuat web sesederhana mungkin.

Materi-materi yang digunakan pada web ini adalah untuk HTML seperti paragraf, heading, hyper link, list, image dan lainnya. Untuk CSS dasar seperti selector, penempatan css, background, float, clearfix, pseudo class, text style. Untuk CSS layouting seperti position, z-index, overflow, box model, display. Untuk CSS3 yang sudah digunakan di halaman ini seperti font-face, web-font generator, transition.

Elemen CSS3 lainnya akan digunakan setelah penulis paham betul tentang CSS Unit. Yaitu elemen vh, vw, vmax dan vmin. Pada halaman ini CSS Unit yang digunakan seperti value, em & rem, calc() dan unit. Setelah penulis menguasai CSS Unit secara mendalam. Penuls akan mempelajari Javascript lalu mengimplementasikan-nya dengan membuat web yang lebih interaktif lagi dan mungkiin akan memakan waktu yang cukup lama untuk mempelajarinya dan memahaminya.
